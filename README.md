# Voting / Polling App

## 📌 Description
A full-stack polling application where users can create polls, vote, and see live results.

## 🚀 Features
- Create polls
- Vote on options
- Live percentage results
- Total vote count
- Delete polls
- Prevent duplicate voting

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express

## ▶️ How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
Open frontend/index.html

## 📊 API
- GET /polls
- POST /polls
- POST /polls/:id/vote
- DELETE /polls/:id
